(constant) COLUMN_MAP

Column definitions within lldpRemEntry.

Key = column number (string, from OID position). Value = [fieldName, isSubtype].

Subtypes must be stored and looked up when the corresponding value column arrives. Since a single table walk returns results interleaved by index (not grouped by column), we collect subtypes into a separate dict and reference them during value decoding.
